# 静默待办 - 使用说明

`静默待办` 是一个放在 Windows 桌面底层的轻量待办窗口，适合长期显示任务而不遮挡正在使用的应用。

最新版可从 [GitHub Releases](https://github.com/thebsf/todolist/releases/latest) 下载 `QuietTodoWidget.exe`（静默待办）。

## 主要功能

- 桌面底层显示、低透明度风格，可调整大小、颜色、字体和透明度。
- 支持待办与已完成清单，任务完成后可在已完成中查看。
- 支持最多四级子任务、折叠/展开、拖拽排序、跨父任务移动和主任务置顶。
- 锁定后仍可切换清单、打开设置、刷新、全部折叠/展开和滚轮浏览；任务编辑、拖动、行内折叠和窗口缩放保持禁用。
- 长列表的窄滚动提示块仅在鼠标悬浮于任务区域时显示。
- 支持与 Obsidian Markdown 清单双向同步，可在设置中选择要展示的 `.md` 待办文件。
- 支持直接点击任务文字进行原地编辑；回车或点到别处保存，Esc 取消，手动选中文字时显示柔和灰色选区。
- 可选开机启动，并可选择多种低对比度主题或自动融入壁纸。
- 可连接 Microsoft To Do，同步任务、完成状态和一级子任务。

## 直接使用

直接运行发布文件：

```text
dist\静默待办.exe
```

程序只会保留一个运行实例，重复双击不会打开多个窗口。

## 常用操作

- 新增主任务：在待办列表末尾的空白输入行输入内容，按 `Enter` 保存。
- 新增子任务：点击任务右侧的 `+`，在新出现的行中输入内容后按 `Enter`。
- 取消空子任务：不输入内容，点击其他地方即可取消。
- 完成任务：点击任务左侧圆形勾选按钮。
- 修改任务文字：解锁后点击任务或子任务文字，在原位置直接编辑。
- 排序任务：解锁时拖动左侧 `≡` 手柄。
- 折叠任务：点击带子任务行中的展开/折叠箭头；顶部图标可一键折叠或展开全部。
- 锁定窗口：点击右上角锁头；再次点击解锁。
- 调整设置：点击右上角 `···`。

## Obsidian 同步

Obsidian 同步默认关闭。在设置的“Obsidian 联动”中可启用/关闭同步，并通过“选择文件”切换到任意 Markdown 待办清单。文件使用标准任务格式：

```markdown
- [ ] 待办事项
  - [x] 已完成子任务
```

你在 Obsidian 中保存文件后，桌面组件会自动更新；在组件中新增、完成、删除或调整子任务后，也会写回 Markdown。任务行尾的 `<!--qtw:...-->` 是用于同步识别的隐藏 ID。

## Microsoft To Do 同步

在设置中填写 Microsoft Entra 应用注册得到的 `Client ID`，登录 Microsoft 账号并选择清单后，即可与 Microsoft To Do 同步。

需要注意：

- Microsoft To Do 官方接口原生支持一级 checklist 子任务。
- 本组件创建的二至四级子任务保存在本机。
- 未连接 Microsoft To Do 时，任务也可完全在本机使用。

## 本地数据位置

程序数据默认保存在：

```text
%APPDATA%\QuietTodoWidget\
```

其中包括设置、本地任务、同步快照和加密保护的登录缓存。
